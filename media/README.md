# Media Assets Directory

This directory contains all media assets for the Paxi Technologies website.

## Folder Structure

```
media/
├── images/          # General images, photos, graphics
├── videos/         # Video files (mp4, webm, etc.)
├── documents/      # PDFs, documents, downloadable files
├── icons/          # Icon files (SVG, PNG)
├── logos/          # Logo files and brand assets
└── favicons/       # Favicon files for the website
```

## Usage Guidelines

### Images
- Use optimized images (WebP, JPEG, PNG)
- Recommended max width: 1920px for hero images
- Recommended max width: 1200px for content images
- Compress images before uploading

### Videos
- Use MP4 format for maximum compatibility
- Consider WebM for better compression
- Keep file sizes reasonable for web delivery
- Provide fallback images for video thumbnails

### Icons
- Prefer SVG format for scalability
- Use PNG for complex icons if needed
- Standard size: 24x24px, 32x32px, 48x48px

### Logos
- Provide multiple formats (SVG, PNG)
- Include light and dark variants if needed
- Maintain aspect ratio

### Favicons
- Generate favicon package (16x16, 32x32, 180x180, etc.)
- Use ICO format for legacy support
- Include apple-touch-icon (180x180)

## File Naming Convention

Use lowercase letters, hyphens, and numbers:
- ✅ `hero-background.jpg`
- ✅ `company-logo.svg`
- ✅ `service-icon-01.png`
- ❌ `Hero Background.jpg`
- ❌ `company_logo.svg`

## Optimization

Before adding files:
1. Optimize images (use tools like TinyPNG, ImageOptim)
2. Compress videos appropriately
3. Use appropriate formats (WebP for images, MP4 for videos)
4. Consider lazy loading for large assets
