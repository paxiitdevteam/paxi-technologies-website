## Optimization

Before uploading:
1. Optimize images using TinyPNG (tinypng.com) or Squoosh (squoosh.app)
2. Target file sizes:
   - Hero images: < 200KB
   - Content images: < 150KB
   - Service card images: < 100KB
3. Use JPG format for photos, WebP if supported

## Fallback

If images are not available, the site will:
- Display gradient backgrounds (already styled)
- Show placeholder areas
- Function normally without images

## Notes

- All images should be professional and relevant to IT services
- Ensure images are properly licensed for commercial use
- Add descriptive alt text when adding images to HTML
- Consider lazy loading for better performance
